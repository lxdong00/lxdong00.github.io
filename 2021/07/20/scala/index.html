2. 基础知识

2.1基本数据类型和变量

  • 字面量
  • 变量
    • val 变量名:数据类型 = 初始值
    • var 变量名:数据类型 = 初始值

2.2 输入输出

  • 输入
  • 输出
    • print
    • println
    • printf
  • 字符串插值机制
    • s
      • $x,替换变量x的值
      • println(s”s=$s”)
    • f
      • $x,替换变量x的值并可以进行格式控制
      • println(f”f=$f%.1lf”)
  • 写文件
  • 读文件

2.3 控制结构

  1. if else
    • 可以赋值
  2. while
  3. for循环：for(变量<-表达式 if条件表达式){语句}
    • for(i<- 1 to 5 by 2)
    • for(i<- 1 to 5 if i%2 == 0)
    • 多个生成器用;隔开
  4. for推导式：for(变量<-表达式 ) yield {语句}
    • 条件复合将变量的值返回出去
  5. 异常处理
  6. 循环控制

2.4 数据结构

  1. 数组Array
    • val arr = new ArrayInt
    • val arr = Array(1,2,3)
    • arr(0)=1
    • 多维
      • val arr = Array.ofDimInt -三行四列
  2. 元组Tuple
    • val tuple = (“1”,2)
    • 输出println(tuple._1)
  3. 容器Collection
    • 列表List：不可变相同类型序列
      • var strList1=List(1,2,3)
      • var strList2 = 2::3::strList1
    • 向量Vector
      • var vec1 = Vector(1,2)
      • var vec2 = vec1:+2
    • Range
      • var r = new Range(1,5,1)
      • 1 until 5
    • 集合set（可变和不可变）
      • scala.collection.mutable/immutable(默认)
      • vat mySet = Set(“123”,”456”)
    • 映射Map（可变和不可变）
      • var univer = Map(“x”->”y”)

3. 面向对象

3.1 类的定义

  • 可见性
    • 无修饰 = public
    • private，本类型可见，嵌套类型可见
    • protected，本类型可见，继承类型可见

3.2 构造器

  • class 类名(var name:String)会自动生成name属性
  • 任何一个构造器都要调用之前已经定义好的辅助构造器或主构造器

3.3 单例对象

  • 格式
  1. 伴生对象
    • 同时定义class A和object A则互为伴生关系
    • 彼此可以访问内部成员
  2. 孤立对象
    • object A没有与它同名的class

3.4 apply方法

  • 当出现-类名(参数)会自动寻找apply方法
  • 函数可以通过-函数名.apply(参数) 以对象的形式调用

3.5 update方法

  • 当对带有括号并包括一到若干参数的对象进行赋值时编译器将调用update方法
    • myArray(0)=12

3.6 unapply

  • 根据对象把参数提取出来

3.7 继承

  1. 抽象类
    • 类用abstract修饰
    • 没有值或没有方法体就是抽象的
  2. 扩展类
    • extends关键字
    • 重载父类的抽象成员override可选，非抽象成员必须要有override
  3. Option类

3.8 特质

  • 使用trait修饰类
  • 混入特质

3.9 模式匹配

  • case 类
    • case class Car(brand:String)
    • 自动生成伴生类

4. 函数式编程

4.1 函数的定义与使用

  • 函数类型
    • (Int)=>Int
  • 函数的“值”
    • (value)=>{value+=1}
  • 匿名函数
    • (num:Int)=>num*2
  • _的使用
    • val counter = (_:Int) + 1 等效于x:Int=>x+1
    • val add = (_:Int) + (_:Int) 等效于(a:Int,b:Int)=>a+b
