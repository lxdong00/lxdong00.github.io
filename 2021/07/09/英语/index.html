1. 简单句

1.1核心构成

  • 主谓
    • 谓语为非及物动词
  • 主谓宾
  • 主系表
    • be -单独出现
    • get become turn go grow -变得
    • look sound smell taste feel -看/听···起来
    • seem appear/keep remain
  • 主谓双宾
  • 主谓宾补

1.2 谓语变化

①时态

时态 过去 现在 将来 过去将来
一般 did(过去式) do/does will do/ be going to do would do/ was/were going to do
进行 was/were doing be doing will be doing ×
完成 had done have/has done will have done ×
完成进行 had been doing have/has doing will have been doing ×
  • 单三：he、she、it、可数名词单数、不可数名词、不定代词（something、nothing、anything、somebody···）
  • 一般现在时可表示永恒

②情态

现在时 过去时
must 无
can could
will would
shall should
may might
  • 右四个可以表示委婉
  • 后加not变否定
  • 情态动态表推测
    • 情态 + do -推测现在
    • 情态 + be doing -推测现在正在进行
    • 情态 + have done - 推测过去

③语态

  • be(被动的时间并考虑单复数) + done(被动的动作)

④否定

  1. 实义动词变否定
    • do/does/did + not + 动词原形
  2. 助动词和情态动词变否定
    • 助动词和情态动词 + not (be动词、情态动词、完成时态中的have等)
    • 遇到多个情态动词和助动词优先处理第一个

1.3 主宾表变化

  • 名词代词
  • doing
  • to do
    • It is adj. (for sb.) to do sth
    • 只能做动宾不能做介宾
    • It takes (sb.) some time to do sth
  • 并列多个

1.4 简单句扩展

①词性角度扩展

  1. 限定词
    • 冠词：a an the
    • 数词
    • 名词所有格
    • 名词
  2. 形容词、副词
    • 形容词跟在名词前后做定语或系动词后做表语，其他副词来修饰
    • 比较级别
      • 原级：as + 形容词或副词原型 + as + 比较对象
      • 比较级：形容词副词比较级 (+ than 比较对象)
      • 最高级：形容词副词最高级 (+ 介词短语表示范围)
  3. 介词短语
    • 可接名词、代词、doing
    • 表时间
    • 表地点
    • 其他

      ②成分角度扩展

  4. 非谓语动词做定语、状语
    • doing 主动
    • done 被动
    • to do 目的
  5. 同位语、插入语

    2.2 名词性从句

    ① 宾语从句

  • 写法
    • 陈述句： Dr.Worm acknowledges + These figures are conservative
      • Dr.Worm acknowledges (that) These figures are conservative
    • 特殊疑问句：We suddenly can’t remember + Where did we put the keys just a moment age（主语提前）
      • We suddenly can’t remember where we put the keys just a moment age
    • 一般疑问句变宾语从句
      • I don’t know + Are other clients going to abandon me, too?（连接词加whether）
      • I don’t know whether other clients are going to abandon me, too.
  • 只有宾语从句的that可以省略

    ② 表语从句

  • 同宾语从句

    ③ 主语从句

  • 同宾语从句
  • that不做成份不可省略
  • 只能用whether不能用if
  • It is done + 主语从句(表达人们对一件事的观点看法)
  • It is adj./n + 主语从句(表达对一件事的评价)

    ④ 同位语从句

  • 抽象名词后

2.3 定语从句

  • 关系词在从句中做宾语时可以省略

  • 关系词都做成份

  • 范围明确 - 非限定性定语从句

    • 关系词不能用that
    • 关系词不可省略
    • 人做宾语必须用whom

  • 限定性定语从句和同位语从句的区别

2.4 状语从句

  • 写法

  1. 时间状语

    • when / while /as
    • before / after
    • since / until
    • as soon as
    • by the time

  2. 地点状语

    • where

  3. 原因状语

    • because
    • since
    • as

  4. 结果状语

    • so…(形容词/副词) that…
    • such…(名词/名词词组) that…
    • so that

  5. 目的状语

    • so that（后边出现情态动词）
    • in order that

  6. 条件状语

    • if
    • once
    • as long as / so long as

  7. 让步状语

    • althouth / though
    • even if / even though

  8. 比较状语

    • than
    • as

  9. 方式状语

    • as
    • as if

2.5 虚拟

  • if虚拟条件句
    • 从句往前推，主句四加一
  • 名词性从句的虚拟
    • 主句表示建议命令要求，从句要虚拟
    • 从句 should + 动词原形(should可以省略)

2.6 倒装

  • 全部倒装
  • 部分倒装
    • 改为一般疑问句
    • 否定副词或词组位于句首
    • only位于句首
    • 虚拟条件句if省略

2.7 强调

  • It is … that
