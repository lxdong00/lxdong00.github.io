1.基本数据类型

1.1 字符串

  • 定义：’ 或 “ 或 ‘’’
  • 获取
    • 下标
    • 切片:[start:end:step] (不包含end所对应的字符)
    • 长度:len(a)
  • 删除：用del()函数
  • %格式控制:
    • “test%-5d%d”%(age,num)
    • “名字:%(name)s,年龄%(age)d”%{“name”:”铁钢”,”age”:15}
    • 输出%时要替换为%%
  • format
    • “名字:{0},年龄{1}”.format(“铁钢”,12)
    • “名字:{n0},年龄{n1}”.format(n0=”铁钢”,n1=12)
  • f
    • f”名字{name},年龄{age}”
    • f”年龄:{78 - 65 =}”
    • f”年龄:{18:#b}” #转换为二进制显示python3.8版本
  • 其他功能
    • .startswith(“开头”)
    • .endswith(“结尾”)
    • .isdecimal()
    • .strip()

      1.2 数字和运算符

  • 运算符:+,-,*,/,%,**,//
  • 整数
  • 浮点
  • 负数:(1-2j)
  • 布尔
  • 二进制：0b111
  • 数据类型转换
    • 转整形:int(a)
    • 转浮点:float(a)
    • 转复数:complex(x,y)
    • 转字符串:str(a)
    • 转二进制:bin(a)
    • 转八进制:oct(a)
    • 转十六进制:hex(a)
    • 十进制数转ASCII码:chr(97)
    • ASCII转十进制数:ord(‘a’)

      1.3 列表

  • []:可变序列、可存储不同类型的成员
  • help([])查看所有方法
  • 可使用in(判断是否包含元素)，del()(方法删除元素)，切片，+进行列表合并(+会改变被赋值的id，extend不会)

1.4 元组

  • ():不可变序列、可存储不同类型成员

1.5 字典

  • {key:value}
