1.基本定理和规则

1.1基本定理

公里 0
交换律 A + B = B + A 　　 A · B = B · A
结合律 (A + B) + C = A + (B + C) 　　(A · B) · C = A · (B · C)
分配律 A + (B · C) = (A + B)·(A + C) 　　A · (B + C) = A · B + A · C
0-1律 A + 0 = A 　　A + 1 = 1 　　A · 0 = 0 　　A · 1 = A
互补律 A + A’ = 1 　　A · A’ = 0
定理 0
定理1 0 + 0 = 0 　　1 + 1 = 1
定理2 A + A = A 　　A · A = A
定理3 A + A · B = A　　A · (A + B) = A
定理4 A + A’B = A + B　　A · (A’ + B) = AB
定理5 A’’ = A
定理6 (A + B)’ = A’ + B’
定理7 A · B + A · B’ = A　　(A + B)(A + B’) = A
定理8 A · B + A’ · C + B · C = A · B + A’ · C 　　 (A + B) · (A’ + C) · (B + C) = (A + B) · (A’ + C)

2.表达式的形式与变换

2.1基本形式

①与-或表达式

若干与项进行或运算的表达式

②或-与表达式

若干积项进行与运算的表达式

2.2标准形式

①最小项性质

  • 仅有一种取值令最小项为1
  • mi · mj = 0
  • 全部最小项相或为1
  • n个变量构成的最小项有n个相邻最小项

    例：F=A’BC’ + A’BC + ABC’ + ABC = m2+m3+m6+m7

②最大项性质

  • 仅有一种取值令最大项为0
  • Mi + Mj = 1
  • 全部最大项相与为0
  • n个变量构成的最大项有n个相邻最大项

    例：F = (A + B + C)(A + B +C’) = M0M1

③最大项和最小项的关系

  • 下标相同的最小项和最大项互为反函数
  • mi和Mj存在互补关系

2.3转换方法

①代数转换法
转换为与或

  1. 将表达式化为一般与或表达式
  2. 反复使用X = X · (Y + Y’)
    转换为或与
  3. 将表达式化为一般或与表达式
  4. 反复使用X = (X + Y)·(X + Y’)

②真值表转换法

将取值为1的情况或起来或将取值为0的情况与起来

2.4逻辑函数化简

①代数化简法
与或表达式化简

  • 并项法：A · B + A · B’ = A
  • 吸收法：A + A · B = A
  • 消去法：A + A’B = A + B
  • 配项法：A · 1 = A 及 A + A’ = 1

②卡诺图
特点

  • n个变量的卡诺图由2^n个小方格构成
  • 几何图形上相邻、相对、相重位置的小方格代表相邻最小项合并规律
  • 合并理论依据：A · B + A · B’ = A
  • 相邻最小项有一个变量互反，可以合并为一项（卡诺圈）
    化简
  1. 做出卡诺图
  2. 圈出全部质蕴含项
  3. 从全部质蕴含项中找出必要质蕴含项
  4. 求函数最简质蕴含项集

    蕴含项：任何一个mi
    质蕴含项：某个卡诺圈没有被其他更大的卡诺圈包含，则该卡诺圈的与项为质蕴含项
    必要质蕴含项：卡诺圈包含了不可能被其他圈包含的方格，该卡诺圈的与项为必要质蕴含项
